# 🧠 Brain Tumor Analytics Dashboard

This project focuses on analyzing brain tumor patient data using an interactive Power BI dashboard. The goal was to turn raw healthcare data into clear insights that help understand tumor distribution, patient demographics, and diagnosis trends.

## 📊 What This Dashboard Shows

- Total number of patients and key health metrics  
- Distribution of different tumor types  
- Most affected brain regions  
- Age group analysis of patients  
- Year-wise trend of diagnoses  
- Interactive filtering based on tumor grade

## 📂 Dataset

Simulated healthcare dataset with over 10,000 patient records, including information on tumor type, grade, age, and affected regions.

---

## 💡 Key Takeaways

- Most cases are concentrated in specific brain regions like the brainstem and cerebellum  
- Certain age groups are more affected than others  
- Tumor distribution varies significantly across grades  
- Trends over time help understand how diagnosis patterns change  

---

## 🚀 How to Use

1. Download the `.pbix` file  
2. Open it in Power BI Desktop  
3. Use filters (like tumor grade) to explore different insights

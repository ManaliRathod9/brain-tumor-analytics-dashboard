# 🧠 Brain Tumor Analytics Dashboard

This project is a healthcare data analytics dashboard built using Power BI to analyze brain tumor patient data.

## 📊 Key Features
- Total patient analysis with dynamic filtering
- Tumor type distribution visualization
- Age group analysis
- Tumor size and regional impact insights
- Interactive filtering by tumor grade

## 📈 Insights Generated
- Identified most affected brain regions
- Analyzed patient trends over time
- Compared tumor severity across grades
- Found most impacted age groups

## 🛠 Tools Used
- Power BI
- Data Cleaning & Transformation
- DAX (Measures)

## 📂 Dataset
Simulated healthcare dataset with 10,000 patient records.
